# Calculator

A simple yet powerful calculator with basic and scientific modes, built with HTML, CSS, and JavaScript.

## Features

*   **Basic and Scientific Modes:** Switch between a simple calculator for everyday math and a scientific calculator for more complex calculations.
*   **Responsive Design:** The calculator is designed to work on different screen sizes.
*   **Keyboard Support:** Use your keyboard to perform calculations quickly.
*   **Error Handling:** The calculator provides feedback for invalid calculations.

## Keyboard Shortcuts

| Key             | Function                                |
| --------------- | ----------------------------------------|
| `0-9`           | Enter numbers                           |
| `+`, `-`, `*`, `/` | Perform basic operations             |
| `.` or `,`      | Enter a decimal point                   |
| `Enter` or `=`  | Calculate the result                    |
| `Shift`, `Escape`, `Delete`, or `C` | Clear the display   |
| `Backspace`     | Delete the last character               |
| `Tab`           | Switch between calculator modes         |

### Scientific Mode Shortcuts

| Key | Function |
| --- | -------- |
| `s` | sin()    |
| `o` | cos()    |
| `t` | tan()    |
| `l` | log()    |
| `n` | ln()     |
| `p` | π (pi)   |
| `e` | e (Euler's number) |
| `r` | √ (square root) |
| `!` | x! (factorial) |
